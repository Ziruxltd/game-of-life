## Run Locally

Clone the project

```bash
  git clone git@github.com:Ziruxltd/game-of-life.git
```

Go to the project directory

```bash
  cd game-of-life
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
